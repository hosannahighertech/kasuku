Hapa tunahifadhi ikoni zote zinazotumiwa na programu hii.
Unaweza badili mwonekano wa ikoni za programu hii kwa kuweka
ikoni nyingine ukizingatia majina na virefusho (extension).
Ukiisha kuweka ikoni utakazo kwa utaatibu tajwa, kimbiza 
amri hii katika terminal au cmd

 python png2c.py *.png >images.h 
 
 Kumbuka lazima Python iwe imepakiwa katika tarakirishi yako
 na iwe katika njia ya sistimu yako (system path)
 
 Kujua kama Python iko vizuri jaribu amri hii
python --version
